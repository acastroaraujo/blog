Singular Value Decomposition

Author

andrés castro araújo

Published

February 28, 2024

Replace * with \(\top\) !

A singular value decomposition (SVD) is a factorization of the form:

\[ \mathbf{A} = \mathbf{U} \boldsymbol{\Sigma} \mathbf{V}^* \]

definition

where

  • \(\mathbf{A}\) is an \(m\times n\) matrix with rank \(r\)

  • \(\mathbf{U}\) is an \(m\times r\) with \(\mathbf{U}^*\mathbf{U}=\mathbf{I}_r\)

  • \(\mathbf{V}\) is an \(n\times r\) matrix with \(\mathbf{V}^*\mathbf{V}=\mathbf{I}_r\)

  • \(\boldsymbol \Sigma\) is a diagonal matrix comprised of singular values (\(\sigma_1, \sigma_2, \dots, \sigma_r\))

  • The singular values are sorted such that \(\sigma_1 \geq \sigma_2 \geq \dots >0\)

The rank k approximation of \(\mathbf{A}\) is given by the following sum:

\[ \mathbf{A} \approx \sigma_1 \cdot \mathbf{u}_1 \mathbf{v}_1^* + \dots + \sigma_k \cdot \mathbf{u}_k \mathbf{v}_k^* \]

The columns of \(\mathbf U\) and \(\mathbf V\) are usually called “left” and “right” singular vectors.

SVD is a bit magical.

  • \(\underset{n \times n}{\mathbf A^\top \mathbf A} = \mathbf V \boldsymbol \Sigma^2 \mathbf V^\top\)

  • \(\underset{m \times m}{\mathbf{AA}^\top} = \mathbf{U \Sigma}^2 \mathbf U^\top\)

This is related to eigen-vector decomposition!

Ever heard of principal components analysis (PCA)? It’s literally SVD if we first center the columns of \(\mathbf A\) such that each one as a mean of zero.

In fact, if you check the documentation for prcomp() you’ll see the following description:

“The calculation is done by a singular value decomposition of the (centered and possibly scaled) data matrix, not by using eigen() on the covariance matrix. This is generally the preferred method for numerical accuracy.”

Similarity with word embedding

  • The columns of V (referred to as right-singular vectors) are eigenvectors of M⁎M.

  • The columns of U (referred to as left-singular vectors) are eigenvectors of MM⁎.
